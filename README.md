# General explanation

Microservice for receiving and displaying exchange rates from the website of the National Bank of the Republic of Belarus (national bank API: https://www.nbrb.by/apihelp/exrates)

# Technologies and frameworks used:

- [Java XX](https://docs.oracle.com/en/java/javase/17/docs/api/)
- [Maven](https://maven.apache.org/)
- [Spring Boot X.X.X](https://spring.io/projects/spring-boot)
- [Spring Data JPA X.X.X](https://spring.io/projects/spring-data-jpa)
- [Spring MVC X.X.X](https://docs.spring.io/spring-framework/docs/3.2.x/spring-framework-reference/html/mvc.html)
- [H2 Database X.X.X](https://www.h2database.com/html/main.html)
